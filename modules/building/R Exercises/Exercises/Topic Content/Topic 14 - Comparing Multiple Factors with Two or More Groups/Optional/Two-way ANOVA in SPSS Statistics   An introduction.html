Introduction

The two-way ANOVA compares the mean differences between groups that have been split on two independent variables. The primary purpose of a two-way ANOVA is to understand if there is an interaction between the two independent variables on the dependent variable. In ANOVA terminology, the independent variables are normally called "factors". The two-way ANOVA is a specific example of a factorial ANOVA and is often referred to as a factorial ANOVA, despite it being only being one such example.

What is required

In order to run a two-way ANOVA, you require the following:

» Two independent variables that are categorical with two or more groups (e.g., gender and employment status).
» One dependent variable that is continuous (e.g., weight).

See the Types of Variables guide for more information, if needed.

What problems can you solve using a two-way ANOVA?

A two-way ANOVA is most often used for the following purpose:

1. Determine if there is an interaction between two factors on a dependent variable

You have undertaken a study where you had two independent variables (e.g., gender and education level) and measured one dependent variable (e.g., interest in politics). You are interested in how gender and education level interact to predict interest in politics. Considered another way, you believe that the effect of education level on interest in politics depends on gender. You can also phrase it the other way, that the effect of gender on interest in politics depends on education level.

Null and alternative hypothesis

There are three main hypotheses to test when conducting a two-way ANOVA: hypotheses for (1) the main effect for the first factor, (2) the main effect for the second factor, and (3) the interaction effect. These are shown below:

Main effects

For factor 'A':

H0: all group means are equal (i.e. µA1 = µA2 = µA3 = ... = µAk)

where µ = population mean and Ak = number of groups in factor 'A'. The alternative hypothesis is:

HA: at least one group mean is different (i.e. they are not all the same)

For factor 'B':

H0: all group means are equal (i.e. µB1 = µB2 = µB3 = ... = µBk)

where µ = population mean and Bk = number of groups in factor 'B'. The alternative hypothesis is:

HA: at least one group mean is different (i.e. they are not all the same)

Interaction effect

The interaction null hypothesis is that the sum of all individual interactions equals 0 (zero). The alternative hypothesis is that the sum does not equal 0 (zero).

An introduction to this tutorial

This tutorial will guide you through the process of running a two-way ANOVA with post hoc tests in SPSS Statistics, and how to interpret and report the results. It will also show you how to test all the assumptions that underlie the two-way ANOVA, how to interpret and report them, and what to do when the assumptions are violated. You will discover that a lot of the time and effort can be spent in testing these assumptions and making any necessary corrections. The repeated validation of your data caused by the testing of assumptions, making any necessary corrections to violations of assumptions, leads to an iterative process, where you will run a test, correct any violations, run the test again, correct any further violations, etc. We explain this repetitious process from the perspective of running a two-way ANOVA analysis in SPSS Statistics and guide you through the many different pathways that you might have to take to arrive at your final result.

Approaches to understanding an interaction

One of the main reasons for running a two-way ANOVA is to establish whether there is an interaction between two independent variables on a dependent variable. Once you have established whether there is a statistically significant interaction or not, there are a multitude of different approaches to following up the result. If there isn't an interaction, you face whether to fit a reduced model or not (explained later on in this guide). Furthermore, if you do have an interaction, there are different options on how to follow-up the interaction. This guide will present one of the more popular options for tackling this problem - simple main effects (SME).

Assumptions & order of testing

For a two-way ANOVA to be able to provide a valid result, the following three assumptions must hold:

1. There are no outliers in any group (or overall).
2. Each group's data (or residuals) is normally distributed.
3. Each group's data (or residuals) has equal variance (called homogeneity of variances).

These assumptions need to be tested before you can run a two-way ANOVA. Fortunately, the two-way ANOVA is fairly "robust" to violations of normality. "Robust" means that the assumption can be violated (a little) and still provide valid results. Therefore, you will often hear of this test only requiring approximately normal data and some argue that data can even be fairly skewed as long as the number of cases in each group is similar. These assumptions will be tested in the following order in this guide:

1. Detect any outliers.
2. Determine if data is approximately normally distributed in each group.
3. Determine if there is homogeneity of variances.

Depending on the presence of outliers or the normality of the data, you will be instructed how to proceed with testing. This could include making corrections to your data or using an alternative statistical test. This will also include the route to take when, and if, any corrections you make still results in the data failing the assumptions of this test.

These choices will be clearly signposted for you as you work through the guide.
